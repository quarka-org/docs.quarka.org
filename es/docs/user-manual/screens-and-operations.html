Screens and Operations

Comprehensive documentation for all QA Advisor plugin screens and their operations.

Overview​

This section provides detailed guides for each screen in the QA Advisor WordPress plugin. Each screen documentation includes:

  • Purpose and functionality - What the screen does and why you'd use it
  • Navigation instructions - How to access the screen
  • Feature explanations - Detailed breakdown of all capabilities
  • Common workflows - Step-by-step guides for typical tasks
  • Configuration options - Available settings and customizations
  • Troubleshooting - Solutions for common issues

Available Screens​

Analytics and Monitoring​

  • Dashboard - Main analytics overview and key performance indicators
  • Realtime - Live visitor activity monitoring and real-time analytics
  • Audience - Visitor demographics, behavior patterns, and user segmentation
  • Acquisition - Traffic sources, campaign tracking, and referral analysis

Content Analysis​

  • Landing Pages - Page performance analysis and optimization insights
  • Top Growing - Trending content identification and growth metrics
  • All Pages - Comprehensive page-level analytics and performance data

Advanced Features​

  • Brains - AI-powered insights and recommendations engine
  • Goals - Conversion tracking setup and optimization tools
  • Heatmaps - Visual analytics and user interaction tracking

Configuration and Support​

  • Settings - Configuration hub for tracking, privacy, and integrations
  • License Activation - Plugin licensing management and activation
  • Help - Support resources, documentation access, and troubleshooting

Getting Started​

If you're new to QA Advisor, we recommend starting with:

  1. Dashboard - Get familiar with the main interface
  2. Settings - Configure your tracking preferences
  3. Audience - Understand your visitors
  4. Heatmaps - Set up visual analytics

Need Help?​

Each screen documentation includes troubleshooting sections, but if you need additional support:

  • 📧 Email Support: support@quarka.org
  • 📖 Getting Started Guide: Installation and Setup
  • 🐛 Bug Reports: GitHub Issues

Choose a screen from the sidebar to learn more about its specific features and capabilities.
