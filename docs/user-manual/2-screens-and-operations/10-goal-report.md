# Goal Report

The Goal Report shows your site results (conversions).

Goals track visitor actions such as reaching a specific page or clicking a button.  
You can also view user behavior data filtered to sessions that achieved the goal.

If you have not created a goal yet, see [How to Set Up Goals](./goal-setup).

*The Goal menu is available only in **Advanced Mode**.*

---

## Goal Trend

The chart at the top shows the trend of goal completions during the selected period.

---

## Goal Status

Each goal displays the following information:

- Completions  
- Goal Value  
- Conversion Rate  

Select a goal title to switch the reports below to that goal’s data.  
Selecting **All Goals** shows the combined data for all goal completions.


---

## Analyze Goal Completion Sessions

When you select a goal, the data below is filtered to sessions that achieved that goal.

You can view information such as:

- Source / Medium  
- Landing Pages  
- Heatmaps  

This helps you analyze **user behavior that led to goal completion**.

---

## Session Recordings

You can also view recordings of sessions that achieved the goal.

Watching real user interactions helps you better understand the behavior leading to goal completion.

---

## Notes

Goals can be created even after data has been collected.

If the required tracking data exists,  
goal completion data can also be viewed for past sessions.
