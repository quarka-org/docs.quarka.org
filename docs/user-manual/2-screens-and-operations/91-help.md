# Help

The Help screen provides support links and environment information.

## Environment Information

Displays the current plugin version, WordPress and PHP versions, memory limit, and other server details.

## Help

- **Document** — Link to the online documentation
- **WordPress Support Forum** — Link to the community support forum

## What's New

Displays recent release information.

## Debug

Displays detailed WordPress environment and plugin debug information.  
This section includes environment-specific details such as plugin log URLs, so **be careful not to post it as-is in public forums**.
