# ヘルプ

ヘルプ画面では、サポート関連のリンクと環境情報を確認できます。

## 環境情報

現在のプラグインバージョン、WordPress・PHP のバージョン、メモリ上限などが表示されます。

## ヘルプ

- **ドキュメント** — オンラインドキュメントへのリンク
- **WordPress公式サポートフォーラム** — コミュニティサポートへのリンク

## What's New

最近のリリース情報が表示されます。


## Debug

WordPress環境の詳細とプラグイン用のデバッグ情報が表示されます。  
プラグインログのURLなど環境固有の情報が含まれるため、**公開の場にそのまま貼り付けないようご注意ください** 。
