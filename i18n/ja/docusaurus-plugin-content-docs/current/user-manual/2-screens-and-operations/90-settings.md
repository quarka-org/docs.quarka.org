# 設定

QA Assistants の設定画面では、プラグインの動作モードやトラッキング方式を管理します。

## タブ構成

設定画面には以下のタブがあります。

| タブ | 内容 |
|------|------|
| **一般設定** | 動作モード・トラッキング方式・データ保存期間の確認（このページで説明） |
| **目標** | コンバージョン目標の設定 → [目標（コンバージョン）の設定方法](./11-goal-setup.md) |
| **Google連携** | Google サービスとの連携（Coming soon） |

---

## 一般設定

一般設定タブには、2つの設定項目と1つの情報表示があります。変更後は **Save** ボタンをクリックしてください。

### Advanced Mode（アドバンスドモード）

| | |
|---|---|
| **種類** | チェックボックス |
| **デフォルト** | OFF |

アドバンスドモードを有効にすると、管理画面の左メニューに詳細レポートが追加されます。

- **Audience**（訪問レポート）
- **Acquisition**（集客）
- **Behavior**（行動 / ページ）
- **Goals**（目標）

:::tip インストール直後は OFF です
まずは基本画面（リアルタイム・レポート・ヒートマップ）に慣れてから、必要に応じて有効にすることをおすすめします。
:::

:::note データ収集量には影響しません
アドバンスドモードは管理画面の表示項目を増やすだけです。ON にしても、サーバーへのデータ収集量が増えることはありません。
:::

---

### Enable Cookieless Tracking（クッキーレス計測）

| | |
|---|---|
| **種類** | チェックボックス |
| **デフォルト** | ON（推奨） |

- **ON の場合（推奨）**: 基本的にクッキーを使わずに訪問者を計測します。Cookie 同意ツールから「同意した」という値が渡された場合にのみ、クッキーを使用した計測に切り替わります。
- **OFF の場合**: 常にクッキーを使用してトラッキングを行います。

---

### Data retention & limits（データ保存期間と PV 上限）

この欄は**読み取り専用**の情報表示です。画面上で直接変更することはできません。

| 項目 | 内容 | デフォルト |
|------|------|-----------|
| **Data retention** | 収集したデータを保持する期間（日数） | 120 日 |
| **Monthly PV limit** | 1か月あたりに収集できるページビュー数の上限 | qa-config.php で定義 |

これらの値を変更するには `qa-config.php` を編集してください。
→ [qa-config.php の設定方法](../1-getting-started/configure-qa-config.md)

:::caution 月間 PV 上限に達した場合
上限に達すると、その月の残りの期間はデータ収集が一時停止されます。翌月になると自動的に再開されます。上限到達時には、管理者メールアドレスに通知メールが送信されます。
:::
