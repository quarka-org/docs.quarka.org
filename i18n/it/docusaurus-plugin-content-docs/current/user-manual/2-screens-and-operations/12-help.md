## FAQ - Domande Frequenti

### Installazione e Setup

#### Come installo QA Advisor?
1. Scarica il plugin dal sito ufficiale o WordPress repository
2. Carica il file ZIP tramite WordPress Admin > Plugin > Aggiungi Nuovo
3. Attiva il plugin
4. Inserisci la chiave licenza se hai una versione premium

#### Quali sono i requisiti minimi?
- **WordPress**: Versione 5.0 o superiore
- **PHP**: Versione 7.4 o superiore
- **MySQL**: Versione 5.6 o superiore
- **Memoria**: Minimo 128MB, raccomandato 256MB
- **Spazio Disco**: Minimo 50MB per dati analytics

#### Come configuro il tracking?
1. Vai in QA Advisor > Impostazioni
2. Abilita "Tracking Automatico"
3. Configura esclusioni se necessario
4. Salva le impostazioni
5. Verifica che il codice tracking sia presente nelle pagine

### Utilizzo e Funzionalità

#### Perché non vedo dati nel dashboard?
- **Verifica Tracking**: Controlla che il codice tracking sia attivo
- **Tempo Elaborazione**: I dati possono richiedere 24-48 ore per apparire
- **Filtri**: Verifica che non ci siano filtri che escludono i dati
- **Cache**: Svuota cache browser e WordPress

#### Come interpreto le heatmap?
- **Rosso**: Aree con alta interazione/attenzione
- **Giallo**: Interazione media
- **Verde/Blu**: Bassa interazione
- **Bianco**: Nessuna interazione registrata

#### Posso escludere il mio traffico?
Sì, vai in Impostazioni > Privacy e aggiungi il tuo IP agli indirizzi esclusi.

### Problemi Tecnici

#### Il plugin rallenta il mio sito
- **Ottimizzazione**: Abilita cache nelle impostazioni
- **Frequenza**: Riduci frequenza raccolta dati
- **Hosting**: Verifica risorse server disponibili
- **Conflitti**: Controlla conflitti con altri plugin

#### Errori JavaScript in console
- **Conflitti**: Disattiva altri plugin temporaneamente
- **Tema**: Testa con tema WordPress predefinito
- **Cache**: Svuota tutte le cache
- **Supporto**: Contatta supporto con dettagli errore

#### Dati mancanti o inconsistenti
- **Configurazione**: Verifica impostazioni tracking
- **Filtri**: Controlla filtri e esclusioni
- **Periodo**: Assicurati di selezionare periodo corretto
- **Sincronizzazione**: Forza sincronizzazione dati
