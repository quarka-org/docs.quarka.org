# Tutte le Pagine

La sezione Tutte le Pagine fornisce un inventario completo delle pagine del tuo sito web con metriche dettagliate, analisi comparative e insights per ottimizzare l'intera struttura del sito.

## Panoramica

L'analisi completa delle pagine è essenziale per:

- **Site Architecture**: Comprendere la struttura completa del sito
- **Content Performance**: Valutare performance di ogni singola pagina
- **SEO Optimization**: Identificare opportunità SEO su tutto il sito
- **User Experience**: Migliorare l'esperienza utente complessiva

## Vista Generale Pagine

### Metriche Principali per Pagina

#### Performance Traffico
- **Visualizzazioni Uniche**: Visitatori unici per pagina
- **Visualizzazioni Totali**: Numero totale visualizzazioni
- **Sessioni**: Sessioni che includono la pagina
- **Utenti Nuovi**: Percentuale nuovi visitatori per pagina

#### Engagement Metrics
- **Tempo sulla Pagina**: Durata media permanenza
- **Frequenza di Rimbalzo**: Tasso abbandono immediato
- **Scroll Depth**: Profondità scroll media
- **Clic Interni**: Clic su link interni dalla pagina

#### Conversion Metrics
- **Tasso di Conversione**: Percentuale conversioni per pagina
- **Valore per Visitatore**: Valore medio generato per visitatore
- **Obiettivi Completati**: Numero obiettivi raggiunti
- **Revenue Attribution**: Ricavi attribuiti alla pagina

### Categorizzazione Pagine

#### Tipi di Pagina
- **Homepage**: Pagina principale del sito
- **Category Pages**: Pagine categoria prodotti/contenuti
- **Product Pages**: Pagine singoli prodotti
- **Blog Posts**: Articoli e post blog
- **Landing Pages**: Pagine di destinazione campagne
- **Utility Pages**: Pagine servizio (contatti, about, etc.)

#### Classificazione Performance
- **Top Performers**: Pagine con migliori performance
- **Average Performers**: Pagine con performance medie
- **Underperformers**: Pagine con performance sotto la media
- **Problem Pages**: Pagine con problemi evidenti
